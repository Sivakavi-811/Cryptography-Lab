## 🚀 Key Features

### 🏛️ Classical Ciphers
- **Vigenere Cipher**: Multi-alphabetic substitution cipher. 🎭
- **Playfair Cipher**: Digraph substitution using a 5x5 matrix. 🎲
- **Affine Cipher**: Linear mathematical transformation cipher. 📈
- **Hill Cipher**: Matrix-based polyalphabetic substitution. 🔢

### ⚙️ Modern Algorithms
- **DES (Data Encryption Standard)**: Symmetric-key block cipher. 💎
- **AES (Advanced Encryption Standard)**: The industry standard for encryption. 🛡️
- **RSA**: Asymmetric-key encryption and digital signatures. 🗝️
- **Diffie-Hellman**: Secure key exchange protocol. 🤝
- **Feistel Cipher**: Structural foundation for many block ciphers. 🧱

### 🔢 Mathematical Foundations
- **Euclidean Algorithm**: GCD and Extended Euclidean calculations. 🧬
- **Fermat's Theorem**: Primality testing and modular inverse. 🧮
- **Euler's Theorem**: Euler's Totient function and modular exponentiation. 📐

### 🛡️ Hash & MAC
- **MD5**: Message-digest algorithm for integrity. 📝
- **CMAC**: Cipher-based Message Authentication Code. ✅

## 💻 Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sivakavi-811/Cryptography-Lab.git
   cd Cryptography-Lab
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python app.py
   ```
   Open `http://127.0.0.1:5000` in your browser. 🌐
